Clorabase is distributed via [JitPack](https://jitpack.io). This guide walks you through adding the dependency and running your first read and write against a GitHub-backed NoSQL database.

## Prerequisites

Before you begin, make sure you have:

- A GitHub account
- A GitHub Personal Access Token with `repo` scope — see the [setup guide](/setup) for exact instructions
- An Android project (minSdk 29+) or a Java project using Gradle

## Add the dependency

<Steps>
  <Step title="Add JitPack to your repositories">
    In your **root** `build.gradle`:

    ```gradle build.gradle
    allprojects {
        repositories {
            // ... existing repositories
            maven { url 'https://jitpack.io' }
        }
    }
    ```
  </Step>
  <Step title="Add the Clorabase dependency">
    In your **module** `build.gradle` inside the `dependencies` block:

    <Tabs>
      <Tab title="Android">
        ```gradle build.gradle
        dependencies {
            implementation 'com.github.Clorabase.Clorabase:Clorabase4A:0.6'
        }
        ```
      </Tab>
      <Tab title="Java">
        ```gradle build.gradle
        dependencies {
            implementation 'com.github.Clorabase.Clorabase:Clorabase4j:0.6'
        }
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Sync your project">
    Run **Sync Project with Gradle Files** in Android Studio, or run `./gradlew build` from the command line.
  </Step>
</Steps>

## Initialize Clorabase

Initialize the Clorabase singleton as early as possible — in your `Application` class or `Activity.onCreate()` for Android, or at the start of your program for Java.

<Tabs>
  <Tab title="Android">
    ```java Application.java
    public class MyApp extends Application {
        @Override
        public void onCreate() {
            super.onCreate();
            try {
                Clorabase clorabase = Clorabase.getInstance(
                    "your-github-username",
                    "ghp_your_token_here",
                    "my-project"
                );
            } catch (Exception e) {
                e.printStackTrace();
            }
        }
    }
    ```
  </Tab>
  <Tab title="Java">
    ```java Main.java
    import clorabase.sdk.java.Clorabase;

    public class Main {
        public static void main(String[] args) throws Exception {
            Clorabase clorabase = Clorabase.getInstance(
                "your-github-username",
                "ghp_your_token_here",
                "my-project"
            );
        }
    }
    ```
  </Tab>
</Tabs>

<Warning>
Never hardcode your GitHub token in production code or commit it to a public repository. If your token is exposed, GitHub will automatically revoke it. Store it securely — for example, in a local properties file excluded from version control.
</Warning>

## Write your first record

<Tabs>
  <Tab title="Android">
    ```java
    // Get the database root collection
    ClorastoreCollection db = clorabase.getDatabase();

    // Prepare data
    Map<String, Object> data = new HashMap<>();
    data.put("name", "Alice");
    data.put("score", 42);
    data.put("active", true);

    // Write to users/user1
    db.collection("users").document("user1")
        .setData(data)
        .addOnSuccessListener(v -> Log.d("DB", "Data saved!"))
        .addOnFailureListener(e -> Log.e("DB", "Error", e));
    ```
  </Tab>
  <Tab title="Java">
    ```java
    // Get the database root collection
    Collection db = clorabase.getDatabase();

    // Prepare data
    Map<String, Object> data = new HashMap<>();
    data.put("name", "Alice");
    data.put("score", 42);
    data.put("active", true);

    try {
        // Write to users/user1
        db.collection("users").document("user1").setData(data);
        System.out.println("Data saved!");
    } catch (ClorastoreException e) {
        e.printStackTrace();
    }
    ```
  </Tab>
</Tabs>

## Read your first record

<Tabs>
  <Tab title="Android">
    ```java
    db.collection("users").document("user1")
        .getData()
        .addOnSuccessListener(data -> {
            String name = (String) data.get("name");
            Log.d("DB", "Name: " + name);
        })
        .addOnFailureListener(e -> Log.e("DB", "Read failed", e));
    ```
  </Tab>
  <Tab title="Java">
    ```java
    try {
        Map<String, Object> data = db.collection("users").document("user1").getData();
        System.out.println("Name: " + data.get("name"));
    } catch (ClorastoreException e) {
        e.printStackTrace();
    }
    ```
  </Tab>
</Tabs>
